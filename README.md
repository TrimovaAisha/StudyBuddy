# StudyBuddy
Тема проекта; платформа онлайн курсов
Описание; StuddyBuddy-это онлайн платформа курсов по программированию.Имеются такие курсы как Python,Java,Ui UX дизайн!
Список страниц; index.html(вход в платформу), aboutus.html(о нашей платформе), catalog.html(каталог с курсами), review.html(отзывы от пользователей)
Функционал JavaScript; 
Выполняется проверка форм на главной странице и в футере (валидация имени и email).
Реализована отправка формы без перезагрузки страницы с выводом сообщения пользователю.
Добавлена фильтрация курсов в каталоге по категориям с помощью кнопок.
Реализована запись на курс с отображением результата на странице.
На странице отзывов обрабатывается форма с radio и checkbox, данные собираются с использованием циклов.
Используются условия if/else, циклы for, обработчики событий click и работа с DOM.
